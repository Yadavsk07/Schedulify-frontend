## Project Structure

```
schedulify-frontend/
│
├── public/
│   └── index.html
│
├── src/
│   ├── api/
│   │   └── api.js
│   │
│   ├── components/
│   │   ├── Navbar.jsx
│   │   ├── Loader.jsx
│   │   └── TimetableGrid.jsx
│   │
│   ├── pages/
│   │   ├── Dashboard.jsx
│   │   ├── ClassTimetable.jsx
│   │   └── TeacherTimetable.jsx
│   │
│   ├── styles/
│   │   └── global.css
│   │
│   ├── App.jsx
│   ├── index.js
│   └── constants.js
│
├── tailwind.config.js
├── package.json
└── README.md
```

```bash
npm install
```
